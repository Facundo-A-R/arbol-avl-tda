# Árbol AVL - Implementación TDA

Implementación completa de un Árbol AVL en C con soporte genérico (void*) y todas las operaciones obligatorias.

## Características

- ✅ Soporte genérico con void* y funciones de comparación
- ✅ Creación, destrucción y gestión de memoria
- ✅ Inserción y eliminación con balanceo automático
- ✅ Búsqueda eficiente
- ✅ Tres tipos de recorridos (In-orden, Pre-orden, Post-orden)
- ✅ Métricas: altura, peso, hojas, factor de balance, nivel
- ✅ Rotaciones simples y dobles
- ✅ Código completamente comentado
- ✅ Programa de prueba completo

## Estructura

- `arbol.h` - Interfaz del árbol AVL
- `arbol.c` - Implementación del árbol AVL
- `main.c` - Programa de prueba con ejemplos

## Compilación

```bash
gcc -o arbol_avl arbol.c main.c -lm
./arbol_avl
```

- [x] Rotaciones (simple y doble)
